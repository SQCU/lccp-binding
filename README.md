
## Setup & Execution

**Prerequisites:**
- Python 3.8+
- A C++ compiler (`g++`, `clang`, or MSVC on Windows)

### Step 1: Install `uv`

If you don't have `uv` installed, you can bootstrap it directly with `pip`. This is the only time you'll need to use `pip` directly.

```bash
pip install uv
```
